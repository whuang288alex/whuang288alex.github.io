Refocus


This is an app that allows the user to refocus an image. The trick here is that it uses a stack of image with different focus as input instead of just one image. When a user chooses a scene point to be “refocused”, the app picks the “best focused” image from the focal stack, and displays the image.

Minirel


This is a group project in which we built a working single-user DBMS that can execute certain SQL queries. It includes 5 layers: the disk I/O layer (UNIX file system), buffer manager layer(using the clock algorithm), heap file layer, query processing layer, and the user interface.

Image Stitcher


This is an app that stitches a collection of photos into one. We accomplished ths by first getting some interest points using the SIFT algorithm, and then using RANSAC to get rid of the outliers. Finally, we calculated the homogrphy between the images based on those SIFT matches, and used backward warping to stitch the images together.
